# JobRecruita Mobile (Expo)

## Getting Started

1. Install dependencies:
	```sh
	npm install
	```
2. Copy `.env.example` to `.env` and fill in required values.
3. Run locally:
	```sh
	npx expo start
	```

## Azure Deployment

For production, build the Expo app and deploy to the App Store/Play Store. You can use Azure App Center for CI/CD and distribution.

## Environment Variables

See `.env.example` for required variables.

## CI/CD

Set up Expo Application Services (EAS) or Azure App Center for automated builds and deployment.

# JobRecruita Mobile

Expo + TypeScript + Expo Router + NativeWind

## Features
- Onboarding, Login/Register
- Dashboard, Candidates, Jobs, Matches
- Swipe-to-match, AI scoring, CV upload
- Animations, skeleton loaders, push notifications

## Getting Started
1. Install dependencies: `yarn install` or `npm install`
2. Start Expo: `expo start`
